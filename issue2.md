### Objective

Configure TanStack Query and Axios to fetch data from the CoinGecko API.

### Tasks

* Create API service for CoinGecko endpoints.
* Configure React Query client with caching strategies.
* Implement error handling and rate limit protections.

### Acceptance criteria

* Axios successfully fetches data from CoinGecko.
* React Query caches the response for 2 minutes.
* UI gracefully handles API errors.
