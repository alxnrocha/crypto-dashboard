### Objective

Implement interactive price history charts for individual cryptocurrencies using Recharts.

### Tasks

* Create `CoinChart` component.
* Fetch historical market data from CoinGecko API.
* Integrate Recharts to render the line chart.
* Implement time range filters (7d, 30d, 1y).

### Acceptance criteria

* Chart renders correctly and is fully responsive.
* Tooltips display accurate price and time data on hover.
* Time range buttons properly refetch or filter the dataset.
