### Objective

Implement the main Dashboard Coin Table to display real-time cryptocurrency data.

### Tasks

* Create `CoinTable` component.
* Integrate with `getMarketCoins` via React Query.
* Display Name, Symbol, Price, 24h Change, and Market Cap.
* Implement a loading skeleton.
* Add responsive design for desktop and mobile.

### Acceptance criteria

* Table renders correctly and fetches data on mount.
* Numbers are formatted as currency and percentages.
* Loading state is visually appealing and matches the design.
