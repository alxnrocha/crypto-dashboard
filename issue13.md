### Objective

Implement a Watchlist page where users can view and manage their favorite cryptocurrencies.

### Tasks

* Create `Watchlist.tsx` page component.
* Configure a new route `/watchlist` in `App.tsx`.
* Update `Sidebar.tsx` navigation to use React Router links for 'Dashboard' and 'Watchlist'.
* Display the favorited coins using the `favorites` state from Zustand.

### Acceptance criteria

* Watchlist page renders correctly and displays only the starred coins.
* Clicking the star icon in the table correctly toggles the item in the watchlist.
* Sidebar links highlight the active route.
